# CFH Automotive Ecosystem: Community Forum Feature List

This document outlines the finalized features for the Community Forum module, covering `ForumPostThread.jsx` (frontend component for viewing/managing forum threads) and `forumRoutes.js` (backend API routes). These support a $50K revenue goal through subscriptions ($5-$20/month) and promoted content fees, fostering discussions on vehicles and CFH services.

## ForumPostThread.jsx
**Path**: C:\CFH\frontend\src\components\forum\ForumPostThread.jsx  
**Purpose**: View and participate in forum threads discussing vehicles, CFH services, and more.

### Free Tier
- View all public threads and posts.
- Create posts/replies with basic formatting (e.g., bold, links).
- Like or mark posts as helpful.
- Report inappropriate content.
- View basic user profiles (username, avatar).
- Subscribe to threads for reply notifications.
- Search thread titles/content (basic).
- Accessibility: Screen reader support, keyboard navigation.
- Error messages for short posts or network issues.

### Standard Tier
- See original post (title, author, timestamp).
- View replies chronologically or nested (2 levels).
- Reply to main post or specific replies.
- Quote previous messages.
- Navigate with breadcrumbs (Forum > Category > Thread).
- Paginate long threads.
- See author details (join date, post count).
- Auctions integration: Discuss auction listings.
- Display user reputation (basic points).

### Premium Tier
- Advanced formatting (tables, video embeds).
- Personalized post signatures.
- Anonymous posting in moderated sections.
- Powerful thread search (filter by author, date).
- No ads in threads.
- Increased private message storage.
- Customized profile options.
- Access exclusive groups (e.g., mechanics).
- Promoted posts for higher visibility.
- Thread analytics: Views, engagement.
- Thread customization: Colors, fonts.
- Earn 50 points/helpful post ($0.10/point).

### Wow++ Tier
- AI summarizes long threads.
- AI suggests best answers for questions.
- Create polls or Q&A posts.
- Earn “Topic Titan” badge for popular threads.
- Live thread updates (no refresh).
- AI-suggested CFH services (e.g., mechanics).
- Join priority “Ask an Expert” Q&A events.
- Tag threads for discovery.
- Blockchain verifies post authenticity.
- Marketplace integration: Promote services.
- Leaderboards for top contributors.
- Monetization: $5-$20/month, promoted fees, $2/API call.

## forumRoutes.js
**Path**: C:\cfh\backend\routes\forum\forumRoutes.js  
**Purpose**: Backend APIs for forum management.

### Free Tier
- View categories and threads.
- Create/view basic posts/replies.
- Like/report posts.
- Subscribe to threads.
- Basic thread search.
- Secure with user login.

### Standard Tier
- Get thread details with nested replies.
- Edit/delete own posts.
- Quote replies.
- View user reputation points.
- Fast, secure API responses (<500ms).

### Premium Tier
- Advanced formatting APIs.
- Manage signatures/anonymous posts.
- Access exclusive group APIs.
- Moderate reported content.
- Promote pinned threads.
- View analytics (popular threads).
- Promoted post APIs.
- Earn 100 points/engagement ($0.10/point).

### Wow++ Tier
- AI thread summaries/best answers.
- Create/manage polls.
- Track points/badges.
- Live thread updates (WebSockets).
- Suggest CFH services.
- Schedule expert Q&A events.
- AI moderation for spam.
- Blockchain post records.
- Marketplace service promotions.
- Monetization: $2/API call supports $50K goal.
