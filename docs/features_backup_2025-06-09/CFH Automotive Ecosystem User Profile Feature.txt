# CFH Automotive Ecosystem: User Profile Feature List

This document outlines the finalized features for the User Profile module, covering `UserProfilePage.jsx` (frontend component for managing user profiles) and `userProfileRoutes.js` (backend API routes). These support a $75K revenue goal through subscriptions ($10-$25/month) and verification fees, centralizing user data and engagement.

## UserProfilePage.jsx
**Path**: C:\CFH\frontend\src\components\profile\UserProfilePage.jsx  
**Purpose**: Manage user info, vehicles, settings, and platform activity.

### Free Tier
- Edit name, email, password, one profile picture.
- Basic notification controls (on/off by category).
- Add up to 3 vehicles (Make, Model, Year).
- View basic activity (last login, bid count).
- Set basic profile visibility (public/private).
- View service order/auction history.
- Add one payment method.
- Basic analytics: Auction participation summary.
- Accessibility: Screen reader support, keyboard navigation.
- Error messages for invalid email/VIN.

### Standard Tier
- Manage phone, address, social links.
- Change avatar, enable two-factor authentication.
- Set language/region preferences.
- Add vehicle details (VIN, nickname, photo).
- Link to valuations/services for vehicles.
- Granular notification settings (email, SMS).
- View detailed order/booking/auction history.
- Real-time activity feed (bids, purchases).

### Premium Tier
- Custom banner, “About Me” section, social links.
- Unlimited vehicles with notes, photos.
- Personal maintenance reminders, service logs.
- Detailed analytics: Spending, bid win/loss.
- Export activity data.
- Advanced privacy/session management.
- “Verified User” badge via identity check.
- Early access to CFH features.
- No ads.
- Increased document storage.
- Customizable profile themes.
- Private profile sections (e.g., bid history).
- Earn 50 points/activity ($0.10/point).

### Wow++ Tier
- AI profile optimization suggestions.
- Interactive “CFH Journey” timeline.
- Gamification hub: Points, badges, leaderboards.
- Set automotive goals (e.g., save for car).
- Secure document vault (titles, insurance).
- Connect with users by interests.
- Exportable vehicle portfolio report.
- AI next-action prompts (e.g., service booking).
- Blockchain data transparency.
- Social sharing of profile/achievements.
- Marketplace integration: Promote services.
- Achievement showcase (e.g., “Top Bidder”).
- Monetization: $10-$25/month, $2/API call, verification fees.
- **CQS**: <1s load time, audit logging.
- **Error Handling**: Retry API failures (1s intervals).

## userProfileRoutes.js
**Path**: C:\cfh\backend\routes\profile\userProfileRoutes.js  
**Purpose**: Backend APIs for profile management.

### Free Tier
- View/edit basic profile: `GET/PUT /users/me`.
- Manage basic notifications: `PUT /users/me/settings/notifications`.
- Add up to 3 vehicles: `POST /users/me/garage`.
- View basic activity/orders: `GET /users/me/activity`.
- Add one payment method: `POST /users/me/payment-methods`.
- Basic analytics: `GET /userActivityFeed`.
- Secure with JWT login.
- **CQS**: Rate limiting (100/hour).

### Standard Tier
- Update phone, address: `PUT /users/me`.
- Manage 2FA, language: `PUT /users/me/settings/account`.
- Add detailed vehicle data: `PUT /users/me/garage/:vehicleId`.
- Detailed history: `GET /users/me/orders`.
- Real-time feed: `GET /userActivityFeed`.
- Fast, secure API responses (<500ms).
- **CQS**: HTTPS, encryption.
- **Error Handling**: 400 invalid inputs, 404 not found.

### Premium Tier
- Custom banners, “About Me”: `PUT /users/me`.
- Unlimited vehicles: `POST /users/me/garage`.
- Analytics/export: `GET /profileAnalytics`.
- Privacy/session APIs: `PUT /users/me/settings/privacy`.
- Verification: `POST /users/me/verify`.
- Advanced notifications: `PUT /users/me/settings/notifications`.
- Custom themes: `POST /updateProfileCustomization`.
- Private data: `POST /restrictProfileData`.
- Earn 100 points/engagement ($0.10/point).
- **CQS**: Redis caching, 99.9% uptime.

### Wow++ Tier
- AI insights: `GET /users/me/ai-profile-insights`.
- Gamification: `GET /users/me/gamification`, `POST /users/me/gamification/redeem`.
- Goals: `GET/POST /users/me/goals`.
- Document vault: `GET/POST /users/me/documents/automotive/:documentId`.
- Networking: `GET /users/me/connect`.
- Journey timeline: `GET /users/me/cfh-journey`.
- AI prompts: `GET /users/me/next-action`.
- Blockchain records: `POST /storeProfileOnBlockchain`.
- Social sharing: `POST /socialShareProfile`.
- Achievement showcase: `GET /users/me/achievements`.
- Marketplace promotions.
- Monetization: $2/API call supports $75K goal.
- **CQS**: <1s response, audit logging.
- **Error Handling**: 429 rate limits, retry timeouts.
