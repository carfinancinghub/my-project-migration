## telematicsRoutes.js
**Path**: C:\cfh\backend\routes\telematics\telematicsRoutes.js  
**Purpose**: Node.js/Express routes for telematics data management.

### Free
- `POST /telematics/data`: Basic device data push.  
- `GET /telematics/vehicles/:vehicleId/current-status`: Delayed location.  
- `GET /telematics/vehicles/:vehicleId/trips`: Last 5 trips.  
- **CQS**: JWT authentication, rate limiting (100/hour).

### Standard
- `GET /telematics/vehicles/:vehicleId/driving-events`: Behavior events.  
- `POST /telematics/devices/register`: Device linking.  
- `PUT /telematics/devices/:deviceId/config`: Device settings.  
- `POST /telematics/users/:userId/geofences`: One geofence.  
- `GET /telematics/users/:userId/alerts`: Basic alerts.  
- Multi-vehicle data: `POST /bulkVehicleTelematics`.  
- Diagnostics: `GET /vehicleDiagnosticsData`.  
- **CQS**: HTTPS, <500ms response, encryption.  
- **Error Handling**: 400 invalid data, 404 not found.

### Premium
- Real-time streaming: `wss://api.cfh.com/telematics/vehicles/:vehicleId/live`.  
- Extended history: `GET /telematics/vehicles/:vehicleId/trips` (90 days).  
- Advanced geofences: Multiple, custom triggers.  
- Crash event: `POST /telematics/vehicles/:vehicleId/crash-event`.  
- Stolen vehicle: `GET /telematics/vehicles/:vehicleId/recovery`.  
- Alert management: `PUT /telematics/users/:userId/alerts/:alertId`.  
- Fleet APIs: Driver assignment, fleet reports.  
- Advanced data: `GET /advancedTelematicsData`.  
- Real-time health: `GET /realTimeVehicleHealth`.  
- Gamification: `POST /telematics/events` (100 points/action).  
- **CQS**: Redis caching, 99.9% uptime.

### Wow++
- `GET /telematics/vehicles/:vehicleId/ai-coaching-insights`: AI tips.  
- `GET /telematics/vehicles/:vehicleId/predictive-diagnostics`: Failure alerts.  
- `GET /telematics/vehicles/:vehicleId/parking-location-ar`: AR parking.  
- `POST /telematics/trips/:tripId/analyze-context`: Contextual analysis.  
- `POST /storeTelematicsDataOnBlockchain`: Blockchain records.  
- `GET /telematics/users/:userId/teen-driver-report`: Teen reports.  
- Insurance UBI: `POST /telematics/insurance-discount`.  
- Gamification: “Safety Star” badge for safe driving.  
- Monetization: $2/API call, contributes to $80K goal.  
- **CQS**: <1s response, audit logging.  
- **Error Handling**: 429 rate limits, retry timeouts.
