# Express Mock App (`backend/__mocks__/index.ts`)

## Purpose

This mock Express app simulates backend endpoints for unit, integration, and E2E testing throughout the CFH project.

## Features

- Minimal server using [Express](https://expressjs.com/).
- Endpoints:
  - `/user/profile` — Mock user profile data.
  - `/arbitrators` — Mock arbitrator data.
  - `/onboarding` — Onboarding flow mock.
  - `/error` — Simulates a 500 error for negative tests.
- Use with Jest, Vitest, or Supertest in backend tests.

## Usage

Import and use as `app` in backend tests:
```typescript
import app from '@mocks/index';
import request from 'supertest';

describe('Mock App', () => {
  it('returns user profile', async () => {
    const res = await request(app).get('/user/profile');
    expect(res.body.success).toBe(true);
  });
});
