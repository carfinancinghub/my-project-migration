# AI Insights API Route Documentation

## Endpoint

- **GET /api/insights**

## Purpose

Exposes platform metrics, forecasts, AI-driven recommendations, and usage stats for dashboards and reporting.

## Query Parameters

| Name       | Type    | Required | Default | Description                                      |
|------------|---------|----------|---------|--------------------------------------------------|
| isPremium  | string  | No       | false   | If true, enables premium metrics/limits          |
| userId     | string  | No       | null    | User identifier for personalized insights        |

## Security

- Uses [helmet](https://www.npmjs.com/package/helmet) for HTTP headers.
- Per-route rate limiting (higher for premium).
- Intended to add auth middleware for premium user gating.
- Validation of all query params with @utils/validateQueryParams.

## Features

- Returns metrics, predictions, and actionable AI tips.
- Rate limits: 120 req/min (Premium), 60 req/min (Free).
- Caches insights in Redis/memory for faster repeated access (planned).
- Typed request/response for API contract stability.
- Logs errors and validation issues with @utils/logger.

On error: { "success": false, "message": "..." }
