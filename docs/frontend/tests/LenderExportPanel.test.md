# LenderExportPanel Test Suite – Documentation

## Overview

Describes **unit and integration tests** for the `LenderExportPanel` React component, which handles CSV export, premium AI negotiation simulation, and user feedback.

## Test Cases

- **Basic Export (Free):**
  - Exports lender data as CSV, validates content, asserts file creation and success UI feedback.
- **Premium Export:**
  - Simulates AI negotiation for premium users (mocks fetch to return negotiation outcome), asserts correct UI update.
  - Tests error handling for negotiation failure.
- **Edge Cases:**
  - Handles empty dataset (exports empty CSV, shows user feedback).
  - Tests large/invalid dataset export edge cases.
- **Accessibility:**
  - Ensures export button has proper ARIA labels.
- **Snapshot:**
  - UI snapshot regression check for major features.
- **Mocks:**
  - Logger, `URL.createObjectURL`, and user feedback (toasts/banners) are all mocked.
