# LoanRecommendationController – Documentation

## Overview
Component for providing advanced, smart loan recommendations to buyers in the CFH Automotive Ecosystem. Supports prioritization, export to CSV, AI reasoning for premium tiers, and accessibility enhancements.

## Main Features
- **Loan recommendation list** with match scores and AI explanations (Premium/Wow++ only)
- **Prioritization**: lowest rate, lowest payment, no down, shortest term
- **Export**: Download recommendations as CSV (disabled if loading/error)
- **Fallback UI**: If no recommendations found
- **Accessibility**: All controls labeled; keyboard accessible
- **Validation**: Input checked via @validation/recommendations.validation
- **Auth**: Token use is placeholder; should be upgraded to context/provider

## Tier-based Features
- **Free**: Basic recommendations and export
- **Premium/Wow++**: AI match score, reasoning, improved export analytics

## API/Service
- Uses `@services/recommendations` for fetching
- Validation with Yup schema
- Test suggestions: RTL/Jest for error, loading, export

## Future Improvements
- **Pagination** for large datasets
- **Better token security**: Use React context/provider
- **Server-side sorting/filtering**

# Lender – Documentation

## Overview
Defines the **Lender** Mongoose model. Supports bulk loan removal, filtering by specialization, unique company/licensing, and premium lender management.

## Main Features
- **Lender info:** company, license, funding limit, approval
- **Specializations:** enum-typed (fico, equity, ira, fleet, refinance)
- **Loan management:** add, remove, removeAll loans from active list
- **Search:** Static for findBySpecialization
- **Indexes:** On user, approved
- **Premium logic:** Support for unique/premium lender features

## Methods/Statics
- `addLoan(loanId)`, `removeLoan(loanId)`, `removeAllLoans()`
- `findBySpecialization(spec)`
- Pre-save: ensures specialization uniqueness
